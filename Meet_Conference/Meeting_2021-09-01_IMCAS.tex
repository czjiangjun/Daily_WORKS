\section{王崇愚~院士}
周光召:~人类的梦想:~实现材料按需设计

实现资源化的合金

\ch{Fe}、\ch{Co}、\ch{Ni}和\ch{Ta}、\ch{W}、\ch{Re}都是只差一个电子，物理性质的差别

“概率幅”中的相位问题
设计的考虑和规则(见\textrm{PPT}第29页)
\begin{itemize}
	\item 周期平移对称性
	\item 元素择优占位规则
	\item 元素分布几何组态及其组合，可视多样性(千量级)材料原子模型
	\item 自旋极化效应
	\item 500原子体系，实施分区并行算法
	\item 合金化元素的线性标度算法
	\item 不同元素的颜色表示
\end{itemize}
计算材料按需设计多层次框架与物性参数关联(流程图)

高温合金物性研究阶段性进展(成果)
\begin{itemize}
	\item 千量级材料设计的理念、理论和调控
	\item \textrm{Green}函数桥接模型
	\item 有限元与分子动力学耦合
	\item 能带分析中的反折叠计算
	\item 时间多尺度研究进展
	\item 多学科集成研究，发展物性可调控的资源化四代高温合金(关键力性蠕变寿命可达>350小时)
\end{itemize}

总能、结合能(自有原子能量差)、结构能(求和为本征值求和)、形成能(只算合金化元素的形成能)、\textrm{Fermi}能、\textit{d}-电子的关联和分布(\textit{d}-电子对称性)和\textrm{LDOS}(选择物理内涵比较多的\textrm{LDOS})

\textcolor{red}{考虑花样(\textrm{patten})}
\begin{itemize}
	\item \ch{Ta}优先占\ch{Al}位，可以考虑占\ch{Ni}位
	\item \ch{W}弥散型分布，考虑主要在$\gamma^{\prime}$，其次在界面，再次在$\gamma$
	\item 沿(110)方向(垂直纸面)\textrm{ABAB}堆垛，当前合金原子处于上面的\textrm{AB}，建议一半原子移动到下面一层的\textrm{AB}层上
\end{itemize}
这样的花样就可以达到上千量级

	\item 初融(温度)、共晶(消除)、拓扑相、枝晶
	\item 热处理工艺
